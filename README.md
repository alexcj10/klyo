# Clone the repository
```
git clone https://github.com/alexcj10/klyo.git
```

```
cd klyo
```

# Install dependencies
```
npm install
```

# Start development server
```
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
